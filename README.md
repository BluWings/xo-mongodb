xo-mongodb
==========

[MongoDB](http://www.mongodb.org) binding for [eXtendend Objects (XO)](https://github.com/buschmais/extended-objects).

License
-------

``xo-mongodb`` is contributed under Apache License 2.0.

Build
-----

Start the Maven build on command line

    mvn clean package
